Instalando Bootstrap

Página do Bootstrap:Bootstrap

Primeira Etapa: Comando para instalar Bootstrap:

npm i bootstrap@5.3.3

Segunda Etapa:

É abrir o arquivo angular.json e adicionar os arquivos CSS e JavaScript do Bootstrap.

Localize as características: styles e scripts e adicione o caminho do Bootstrap respectivamente:

  • "node_modules/bootstrap/dist/css/bootstrap.min.css"
  • "node_modules/bootstrap/dist/js/bootstrap.min.js"

Por exemplo - arquivo: angular.json

"styles": [ "@angular/material/prebuilt-themes/indigo-pink.css", "src/styles.css", "node_modules/bootstrap/dist/css/bootstrap.min.css"],

"scripts": ["node_modules/bootstrap/dist/js/bootstrap.min.js"]
